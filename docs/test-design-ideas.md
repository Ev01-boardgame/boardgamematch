# 測驗類型構想：人格 × 桌遊偏好研究

> 目標：分類人，研究性格與桌遊喜好的關係。  
> 情境：公開釋出、約 24 題、5–15 分鐘、預期 500+ 樣本。

---

## 一、你現有設計的對齊

- **24 題四階段漏斗**（社交與組織 → 決策與思考 → 美學與體驗 → 極端情境）很適合當「主測驗」，一組題同時餵給多種解釋（人格型 + 偏好標籤）。
- **十大桌遊人格**（派對主揪、交際王、和事佬、開心果、好勝軍師、桌遊純血、美感派、故事派、收藏家、混沌玩家）可當「人格面」的輸出；同一份 24 題結果再透過**標籤池 + 解釋濾鏡**產出 MBTI 風格、遊戲推薦等，技術路線清楚。

下面在「人格分類」與「桌遊喜好分類」兩大類底下，各給多種**測驗類型想法**，方便你選做哪些測驗、以及如何和 24 題主測驗搭配。

---

## 二、人格分類測驗（用來「分類人」）

| 類型 | 說明 | 研究用途 | 題數建議 |
|------|------|----------|----------|
| **桌遊情境人格（10 型）** | 你提的 Host / Warhawk / Collector… 等，在「玩桌遊時」的行為與動機。 | 直接對應桌遊行為與偏好，易做交叉分析。 | 主測驗 24 題即可產出。 |
| **MBTI 風格（16 型）** | 四維度 E/I、S/N、T/F、J/P，概念接近 16 型。 | 大眾熟悉、利於對外溝通與社群分享。 | 可共用 24 題或獨立 16–24 題。 |
| **大五人格（簡版）** | 外向性、開放性、嚴謹性、親和性、情緒敏感度。 | 學術常用、信效度好，便於與文獻對接。 | 獨立 15–25 題（每向度 3–5 題）。 |
| **社交風格** | 主揪型 / 參與型 / 觀察型 / 獨行型。 | 解釋「誰揪團、誰跟團、誰在邊緣」。 | 可從 24 題第一階段抽維度或獨立 6–8 題。 |
| **決策風格** | 理性分析 / 直覺 / 風險愛好 / 風險規避 / 從眾。 | 與策略遊戲、賭博機制、長考容忍度相關。 | 可從 24 題第二階段抽或獨立 6–8 題。 |
| **競爭 vs 合作傾向** | 競爭導向 / 合作導向 / 平衡型。 | 直接對應合作遊戲、對抗遊戲偏好。 | 可從 24 題第一階段 + 極端情境抽或獨立 4–6 題。 |
| **規則態度** | 嚴守規則 / 彈性詮釋 / 享受打破或 house rule。 | 與「桌遊純血」vs「混沌玩家」相關。 | 可從 24 題極端情境抽或獨立 4–6 題。 |
| **動機維度** | 成就 / 社交 / 探索 / 沉浸 / 放鬆。 | 解釋「為什麼來玩桌遊」，與遊戲類型選擇相關。 | 可獨立 10–15 題或嵌進 24 題。 |

**實作建議**  
- 主軸：**一份 24 題主測驗**產出「10 型桌遊人格」+ **標籤池**（社交性、競爭性、策略性、混亂性、沉浸性、照顧性…）。  
- 同一標籤池再套**濾鏡**：MBTI 風格、遊戲推薦。  
- 若要做大五或決策風格等，可：  
  - 從 24 題中對應題目映射出維度，或  
  - 另做**短版獨立測驗**（各 6–12 題），與主測驗一起施測或分階段釋出。

## 三、桌遊喜好分類測驗（用來「分類偏好」）

這類測驗專門量「喜歡什麼樣的桌遊」，結果可與人格變項做相關或迴歸。

| 類型 | 說明 | 研究用途 | 題數建議 |
|------|------|----------|----------|
| **遊戲類型偏好** | 派對 / 策略 / 合作 / 劇情 / 抽象 / 家庭 / 派對推理…。 | 最直觀的「喜歡哪一類遊戲」。 | 8–12 題（多選或強迫排序）。 |
| **機制偏好** | 工人放置、手牌管理、擲骰、談判、隱藏身份、板塊拼放、競標…。 | 與人格中的策略/社交/隨機容忍度對接。 | 10–15 題（同意程度或偏好排序）。 |
| **主題偏好** | 奇幻 / 科幻 / 歷史 / 經濟 / 動物 / 抽象 / 恐怖…。 | 與開放性、沉浸感、美感需求相關。 | 6–10 題。 |
| **遊戲重量** | 輕策 / 中策 / 重策（或 1–5 自評）。 | 與策略性、學習意願、時間投入相關。 | 可 3–5 題或 1 題自評 + 題目交叉驗證。 |
| **時長偏好** | 快節奏 15–30 分 / 標準 1h / 長局 2h+。 | 與耐心、聚會型態相關。 | 3–5 題。 |
| **玩家互動強度** | 零互動 / 間接互動 / 直接對抗 / 談判結盟。 | 與社交性、競爭性、和事佬型高度相關。 | 6–8 題。 |
| **隨機性容忍度** | 低（偏純策略）↔ 高（接受骰運、抽牌）。 | 與桌遊純血 vs 混沌玩家、策略性相關。 | 4–6 題。 |
| **學習曲線偏好** | 喜歡複雜規則 / 喜歡簡單上手 / 看情境。 | 與開放性、嚴謹性、時間成本相關。 | 4–6 題。 |
| **重開性 vs 體驗型** | 同一款反覆鑽研 vs 玩過就換、重體驗。 | 與收藏家、探索欲、成就動機相關。 | 4–6 題。 |
| **具體遊戲偏好** | 「以下遊戲你喜歡哪些？」勾選或評分。 | 可事後對遊戲做標籤（機制/主題/重量），再與人格做關聯。 | 10–20 題（可隨版本擴充）。 |

**實作建議**  
- 第一波：**類型 + 機制 + 互動強度 + 隨機性**各做一個短模組（各 4–8 題），總計約 20–30 題，可拆成「人格 24 題 + 偏好 20 題」兩部分，或合併成一份 40+ 題（需說明較長）。  
- 進階：用「具體遊戲偏好」當效標，驗證人格與偏好的預測力。

## 四、多測驗如何配合「性格 ↔ 桌遊喜好」研究

1. **每人一組 ID**：主測驗 24 題 → 產出 **10 型人格** + **標籤池**；可選做 **MBTI 濾鏡**、**大五短版**等。  
2. **同一人**再做 **桌遊偏好模組**（類型、機制、互動、隨機性等）→ 得到 **偏好向度/類別**。  
3. **分析**：  
   - 10 型 × 遊戲類型交叉表  
   - 標籤池（社交性、競爭性…）與機制偏好、重量、時長的相關/迴歸  
   - 若有大五：大五 × 偏好；若有多種人格測驗：比較哪種人格架構對偏好預測力最好  

這樣你就同時有「人格分類」與「桌遊喜好分類」，且能明確對接你現有的 24 題架構與 10 型系統。

---

## 五、建議的優先順序（對齊你的 QA）

- **個人興趣探索 + 統計分析 + 一般大眾 + 約 24 題、5–15 分鐘、公開釋出**  
  - 優先做：  
    1. **主測驗 24 題**（四階段漏斗）→ 10 型 + 標籤池 + MBTI 濾鏡。  
    2. **桌遊偏好**：先做「遊戲類型偏好」+「互動強度」+「隨機性/策略感」各 4–6 題，總約 12–18 題，與主測驗分開或接在後面。  
  - 第二波再考慮：大五短版、具體遊戲偏好列表、推薦系統濾鏡。

若你接下來要實作「24 題四階段題目」或「標籤池 + 濾鏡」的具體題目與計分規則，可以指定先做哪一塊（例如：只做第一階段 6 題對應 10 型的計分），我可以依你現有 `personality-test.html` 的結構幫你設計題目與邏輯。
